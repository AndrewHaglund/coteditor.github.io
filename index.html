features

CotEditor is a lightweight plain-text editor for OS X with the following major features:

syntax coloring
Colorize 17 pre-installed major languages like HTML, PHP, Python, Ruby or Markdown. You can also create your own settings.
powerful find and replace
Super powerful find and replace using OgreKit framework. Regular expression search is of course supported.
outline menu
Extract specified lines with the predefined syntax and you can jump to the corresponding line.
split view
Split a window into multiple panes to see different parts of your document at the same time.
scriptable
Make your own macro in your favorite language, whether it is Python, Ruby, Perl, PHP, UNIX shell or AppleScript.
incompatible characters
Check and list-up the characters in your document that cannot be converted into the desired encoding.
keybindings
Fully customizable keybindins.
CJK language friendly
Estimate various file encodings accurately and handle its line height correctly.
